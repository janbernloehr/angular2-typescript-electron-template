# README #

### What is this repository for? ###

* Providing a template for using angular2 and typescript with electron.


* built with :heart: and [Visual Studio Code](https://code.visualstudio.com)

### Prerequirements ###

* node

### Getting started

* cd to project dir
* `npm install`
* `npm run build`
* `npm run electron`

for continuous rebuilding type

* `npm watch`
